# Copo Social 🍸
Rede social efêmera para eventos e baladas. Os usuários conectam seus copos NFC, criam perfis e se encontram na festa.

## Como rodar localmente
```bash
python server.py
```
Acesse: http://localhost:8000

## Tecnologias
- Python (http.server)
- HTML/CSS/JS puro (SPA)
- Zero banco de dados (estado em memória)

## Páginas
- `/` — App do usuário (celular)
- `/telao` — Telão da balada (TV/projetor)
- `/moderacao` — Painel do moderador
